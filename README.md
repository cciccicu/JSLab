# JSLab

这是一个 Vela 快应用，适用于 Xiaomi Smart Band 9 Pro

通过此快应用，你可以在手环上编写和运行JavaScript

## Feature：

- 实现代码存储

- 实现脚本运行

- 调整了喵喵输入法，使其更适用于JS编写

- 添加了光标支持

- 重新实现console.log

- 实现脚本调用系统API

## 快速上手

### 1. 开发

```
npm install
npm run start
```

### 2. 构建

```
npm run build
npm run release
```

### 3. 调试

```
npm run watch
```
